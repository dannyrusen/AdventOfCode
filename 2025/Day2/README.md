# Day2 Project

## Overview
This project currently contains an empty `part1.py` module and a pytest test scaffold.

## Structure
- `part1.py`: (empty) Add your implementation here.
- `tests/test_part1.py`: Placeholder tests verifying the module imports.
- `pytest.ini`: Pytest configuration (quiet output, report extra info for skipped/xfailed tests).
- `requirements.txt`: Python dependencies (pytest).

## Setup
Create / activate a virtual environment if not already:
```
python -m venv .venv
source .venv/bin/activate
```

Install dependencies:
```
pip install -r requirements.txt
```

## Running Tests
```
pytest
```

### Adding Tests
Replace the placeholder test in `tests/test_part1.py` with real assertions as you add functionality to `part1.py`.
